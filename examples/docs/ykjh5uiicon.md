<br/>
<br/>
<br/>

:::demo
```html
<ykj-h5-ui-icon name='iconicn_m_item' />
```

<br/>
<br/>
<br/>

## API

### Icon props

| 参数  | 说明 | 类型 | 默认 |
| ------------- | ------------- | ---- | ---- |
| name  | icon名称 |  string    | - |
| className  | class名称 |  string    | - |
| styleObj  | style对象 |  object    | - |
