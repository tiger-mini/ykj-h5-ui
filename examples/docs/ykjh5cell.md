## API

### Cell props

| 参数  | 说明 | 类型 | 默认 |
| ------------- | ------------- | ---- | ---- |
| title  | 左侧标题 |  string    | - |
| content  | 右侧内容 |  string    | - |
| is-link  | 是否展示右侧箭头并开启点击反馈 |  boolean    | false |
